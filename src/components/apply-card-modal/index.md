# ApplyCardModal 申请卡弹窗

## 基本使用

```tsx
import React from 'react';
import { ApplyCardModal } from 'karbon-components';
import { Button } from 'antd';
export default () => {
  return <Button onClick={() => ApplyCardModal()}>点击我</Button>;
};
```
